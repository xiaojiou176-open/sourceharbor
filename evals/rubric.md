# Evaluation Rubric

## Regression Gate

A change is a regression if it lowers the agreed pass rate or drops one of the required dimensions below the baseline policy.

## Dimensions

- factuality
- coverage
- citation hygiene
- failure honesty

The point is not to create fake precision. The point is to keep evaluation expectations inspectable.

In practice, this rubric sits behind the reader-first public story:

- public docs explain the product and proof boundary
- the rubric explains how regression is judged once a maintainer is already checking the engineering lane
